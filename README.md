A high-fidelity Amazon Homepage Clone developed to demonstrate advanced front-end layout techniques.
This project focuses on replicating the complex UI/UX of a major e-commerce platform using clean, semantic code. 

✨ Key Features-
Dynamic Navigation Bar, a "Deliver to India" location selector, and a multi-category search bar.
Interactive UI Elements: Includes stylized dropdown menus for language and department selection, and FontAwesome icons for a realistic look.
Hero & Shop Sections: A large-scale visual banner followed by a 4-column product grid covering categories like Electronics, Home, and Fashion. 
Professional Footer: A comprehensive multi-level footer. 

🛠️ Tech Stack- 
HTML5: Used for structured, semantic document markup in index.html. 

CSS3: Implemented in style.css using Flexbox for precise alignment and custom hover effects to enhance interactivity. 

FontAwesome: Integrated to provide high-quality scalable vector icons throughout the interface.

📁 Project Structure -
index.html: The core structure and content of the homepage. 
style.css: The stylesheet containing all layout rules and visual designs. 
images/: Local directory containing all product, logo, and banner assets.
